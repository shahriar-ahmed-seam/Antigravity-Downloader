Drop your hero image in THIS folder and name it one of:

  hero.jpg   hero.jpeg   hero.png   hero.webp

It is picked up automatically by the app's hero banner (see Hero.kt).
If no image is present, the app falls back to an indigo gradient banner.

Recommended size: 1080 x 540 (2:1), landscape, dark-ish so the white
title text stays readable.
